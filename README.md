
# Laravel Stisla  

Sample implementation of [Stisla](https://getstisla.com/) admin template on [Laravel](https://laravel.com)

## Installation

``` bash
# Install composer dependencies
composer install

# Install npm dependencies
npm install

# Build assets
npm run dev

# Copy .env.example
cp .env.example .env

# Generate app key
php artisan key:generate

# Run
php artisan serve
```
